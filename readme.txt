You can create BOOK-like post (divided in chapters)

== Description ==

Make a single post(or etc..) and divide it in Multiple pages, like a chaptered book.

>  (P.S.!!  This is a LIGHTWEIGHT (NON-SLOWING) Plugin.  See other <a href="http://bitly.com/MWPLUGINS" target="_blank" >MUST-HAVE PLUGINS</a> for everyone. )


== Installation ==

1. Download and extract archive to `wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.
